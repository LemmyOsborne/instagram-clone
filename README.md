# Instagram Clone

![Preview](/public/images/profile-preview.png)

Initiated using Vite -> Client-side rendered app featuring an online database, authentication provided by Firebase.

## Tech Stack

-   React
-   React Router
-   Vite
-   TypeScript
-   Firebase
-   Styled-components

## Architecture

-   src
    -   components
    -   constants
    -   context
    -   firebase
    -   helpers
    -   hooks
    -   interfaces
    -   pages
    -   services (firebase functions will live here)
    -   styles
        -   global-styles
